# LangGraph Legacy Reference

This page is a historical and legacy reference for earlier LangGraph-related documentation. It is not current onboarding, not the canonical Voyage workflow, and not evidence that Voyage provides a current LangGraph runtime.

Voyage Framework is a local Project Knowledge OS / Development Memory System for structured development workflows, task memory, context packaging, audit logs, and external AI tool handoff.

The current canonical workflow is based on canonical `task.yaml` input, `voyage tasks` task-state commands, `voyage sync` context workflows, and deterministic prompt packages for manual transfer to external AI tools.

Legacy LangGraph material, including previous tutorial-generation wording such as `voyage docs tutorial <correlation_id>`, is historical and non-canonical. It is not part of the v4.1/v4.2 core and does not define a v4.3 runtime layer. Voyage does not call models or providers, execute agents, provide runtime orchestration, or act as a LangGraph, CrewAI, or AutoGen replacement.
